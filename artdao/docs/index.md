# ArtDAO Documentation

Welcome to the documentation for the ArtDAO project. This documentation provides comprehensive technical details about the ArtCommission and ArtDAO smart contracts.

## Core Components

### ArtCommission Contract

The ArtCommission contract facilitates NFT-based art commissions between clients and artists, managing the full commission workflow including proposal, funding, artwork submission, and final asset exchange.

[View ArtCommission Documentation →](/art-commission/overview)

### ArtDAO Contract

The ArtDAO contract governs membership, dispute arbitration, and treasury decision-making within the art commission ecosystem through DAO NFT ownership and weighted voting.

[View ArtDAO Documentation →](/art-dao/overview)
